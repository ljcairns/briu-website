FAQ

Common questions.

Who is this for?

+

Founders and teams at businesses doing $1M-$50M in revenue who want to deploy AI agents without hiring an internal AI team. You don't need technical expertise — you need a clear workflow you want to improve.

What tech stack do you use?

+

We build on OpenClaw, an open-source agent framework. Models from Anthropic, OpenAI, and others depending on the task. Your agents connect to your existing tools — Gmail, Slack, HubSpot, Salesforce, Google Calendar, and 100+ more.

Who owns the code and data?

+

You do. Every agent, skill, and configuration file belongs to you. Your API keys, your infrastructure. If you stop working with us, everything keeps running.

How do API costs work?

+

You pay model providers directly with your own API keys. We don't add a margin to your token spend. API costs for most agents start at $2-5/day, plus a platform subscription (~$200/mo) shared across all your agents. We set budget ceilings so there are no surprises.

Do you replace employees?

+

No. We automate the repetitive parts of your team's work so they can focus on judgment, relationships, and strategy. The goal is leverage, not headcount reduction.

How fast can we start?

+

A Founder Kickoff takes one session. You'll leave with a working agent deployed on your infrastructure and a written plan for what to build next. Most clients have their first agent running within a week.
